När du skapar ditt Java-projekt i Eclipse:

  1. Markera Create separate source and output folders
  2. Klicka på knappen Next
  3. Under Details, klicka på länken Link additional source
  4. Klicka på knappen Browse
  5. Leta dig fram till mappen javapaket (finns i zip filen).
  6. Klicka på knappen OK i filväljaren
  7. Klicka på knappen Finish i Link Source
  8. Klicka på knappen Finish i New Java Project

Om du redan har skapat ett projekt, blir det lite krångligare att lägga till katalogen med klassfilerna:

  1. Hgerklicka på projektet, välj Properties
  2. Välj Java Build Path
  3. Klicka på tabben Libraries
  4. Klicka på knappen Add Class Folder
  5. Klicka på knappen Create New Folder
  6. Klicka på knappen Advanced
  7. Aktivera kryssrutan Link to folder in the file system
  8. Klicka på Browse
  9. Leta dig fram till mappen javapaket (finns i zip filen).
  10. Klicka på knappen OK i filväljaren
  11. Klicka på knappen OK i New folder
  12. Klicka på knappen OK i Class Folder Selection
  13. Klicka på knappen OK i Properties-rutan
